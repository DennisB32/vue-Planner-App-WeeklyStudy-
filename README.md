# vue-planner-app
## Step 1.
Install Node.js https://nodejs.org/en/download

### Step 2. Windows
```
npm install -g yarn
```

##### - Step 2. Mac
```
brew install yarn 
```

### Step 3. Inside the Projekt folder (root)
```
npm install
```

### Step 4. Inside the Projekt folder (root)
```
npm run serve
```
